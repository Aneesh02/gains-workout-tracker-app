Getting started

How to use Gains

🏋️ Logging a workout

  1. Tap the Workout tab (centre of the bottom bar).
  2. Tap Start Empty Workout or choose one of your saved templates.
  3. Tap Add Exercise and search for an exercise — over 300 are built in, or create your own.
  4. Type the weight and reps using the custom keypad that slides up.
  5. Tap the checkmark to mark a set complete. The row turns green and the rest timer starts automatically.
  6. Tap Finish (top right) when you're done. PRs are flagged automatically.

⌨️ Entering weight & reps

  1. Tap a weight or reps field — the custom keypad slides up.
  2. Use − / + to increment by 0.5 kg or 1 rep without retyping.
  3. Tap NEXT to jump to the next field in the row.
  4. The PREVIOUS column shows your last session's data — tap the checkmark to auto-fill it.

⏱️ Rest timer

  1. The timer starts automatically after you complete a set.
  2. Use the −30s / +30s buttons to adjust on the fly.
  3. Tap the timer bar to change the default rest duration for that exercise.
  4. The timer rings when rest is over.

📊 RPE (Rate of Perceived Exertion)

RPE is optional — it tracks how hard a set felt on a scale of 6–10.

  1. Tap the RPE key on the custom keypad to switch to the RPE picker.
  2. Select a value from 6 to 10 in 0.5 increments.
  3. RPE is shown in workout history and exported to CSV.

📚 Exercise library

  1. The Exercises tab lists all exercises sorted by how often you do them.
  2. Use the search bar or filter by muscle group, movement type (Push/Pull), or category (Compound/Isolation).
  3. Tap any exercise to see your progress chart — e1RM trend, volume, and full set history.
  4. Long-press a custom exercise to edit or delete it.

📚 Workout history

  1. The History tab shows all past sessions grouped by date.
  2. Tap any session to see full details — every set, weight, and PR.
  3. Tap the edit icon inside a session to correct weights, reps, or notes.
  4. Swipe left on a session card to delete it.

🏆 Finishing a workout

  1. Tap Finish — you'll see a summary with duration, total volume, and any new PRs.
  2. A star rating reflects workout quality based on set completion and volume vs your average.
  3. Tap Save as Template to reuse this workout layout later.
  4. The workout is synced to GitHub automatically if you have it connected.

🔔 Smart reminders

  1. Go to Profile → Settings → Reminders and toggle reminders on.
  2. Grant notification permission when prompted.
  3. Tap the tile to pick your preferred reminder time.
  4. Gains sends up to 3 smart notifications per day — morning nudge (9 AM), your chosen time, and an evening wrap-up. Content adapts to whether you've trained yet.

🐙 GitHub sync setup

  1. Go to Profile → Settings → Connect GitHub.
  2. Tap Open GitHub → Create Token — create a fine-grained personal access token with Contents: Read and Write on one repo.
  3. Paste the token back into Gains and tap Validate.
  4. Pick an existing repo or tap Create new repo. Every workout syncs as markdown from now on.

📤 Exporting your data

  1. Go to Profile → Settings → Export Workouts.
  2. A CSV is shared via your phone's share sheet — save to Google Drive, email it, or open in any spreadsheet app.
  3. Includes every set: date, workout name, exercise, weight, reps, RPE, and notes.

🔄 Trying the app with sample data

  1. On the History tab, tap Load sample data if it's empty.
  2. 10 pre-built sessions (Push, Pull, Legs over 5 weeks) populate all charts, PRs, and metrics instantly.
  3. When ready to go real, go to Profile → Settings → Reset all data.

⚙️ App preferences

  1. Sounds — toggle rest timer bell and PR chime under Profile → Settings → Sounds.
  2. Keep screen on — prevents sleep during active workouts. Toggle under Profile → Settings → Keep screen on during workout.
  3. Day starts at — controls when the streak rolls over. Default 4 AM.

🐛 Reporting a bug

Go to Profile → Settings → Support → Report a bug. This opens a pre-addressed email — describe what happened and tap Send.
