Getting started

How to use Gains

🏋️ Logging a workout

  1. Tap the Workout tab (centre of the bottom bar).
  2. Tap Start Empty Workout or choose one of your saved templates.
  3. Tap + Add Exercise and search for an exercise — over 300 are built in, or create your own.
  4. For each set, type the weight and reps using the custom keypad that slides up.
  5. Tap the checkmark to mark a set complete. The row turns green and the rest timer starts automatically.
  6. Repeat for all sets and exercises.
  7. Tap Finish (top right) when you're done. Your workout is saved and any new personal records are flagged.

⏱️ Rest timer

  1. The timer starts automatically after you complete a set.
  2. Use the −30s / +30s buttons to adjust on the fly.
  3. Tap the timer display to change the default rest duration for that exercise.
  4. The timer rings when rest is over — you can silence it in your phone's settings.

📊 RPE (Rate of Perceived Exertion)

RPE is optional — it lets you track how hard a set felt on a scale of 6–10.

  1. Tap the RPE button (small label at the end of a set row) to open the picker.
  2. Select a value from 6 to 10 in 0.5 increments. Swipe the row if all options don't fit.
  3. RPE is shown in your workout history and exported to CSV.

📚 Workout history & editing

  1. The History tab shows all past sessions grouped by date.
  2. Tap any session to see full details — every set, weight, and PR.
  3. Tap the edit icon inside a session to correct weights, reps, or notes after the fact.
  4. Swipe left on a session card to delete it.

📈 Progress & metrics

  1. Tap the Metrics tab (far right) for an overview of your training trends, volume, consistency score, and push/pull balance.
  2. Tap any exercise in the Exercises tab to see its progress chart — volume, estimated 1RM, and full set history.
  3. Your all-time Personal Records are listed on the Profile tab.

🔔 Smart reminders

  1. Go to Profile → Settings → Reminders and toggle reminders on.
  2. Grant notification permission when prompted.
  3. Tap the tile to pick your preferred reminder time.
  4. Gains sends up to 3 smart notifications per day — one fixed morning nudge (9 AM), one at your chosen time, and one evening wrap-up after 8:30 PM. Content adapts to whether you've worked out yet: pre-workout messages nudge lagging muscles or flag streak risk; post-workout messages suggest recovery and tomorrow's targets.

🐙 GitHub sync setup

  1. Go to Profile → Settings → Connect GitHub.
  2. Tap Open GitHub → Create Token — this takes you to github.com to create a fine-grained personal access token.
  3. Set token type to Fine-grained, expiration to your preference, and grant Contents: Read and Write on a single repo.
  4. Copy the token, paste it back into Gains, then tap Validate.
  5. Pick an existing repo or tap Create new repo — the app creates it automatically.
  6. From now on, every completed workout syncs as a markdown file to your repo.

📤 Exporting your data

  1. Go to Profile → Settings → Export Workouts.
  2. A CSV is generated and shared via your phone's share sheet — save it to Google Drive, email it, or open it in any spreadsheet app.
  3. The CSV includes every set: date, workout name, exercise, weight, reps, RPE, and notes.

🔄 Trying the app with sample data

  1. Go to the History tab — if it's empty, tap Load sample data. The same button appears as a banner on the Workout tab for new users.
  2. 10 pre-built sessions (Push, Pull, Legs over 5 weeks) are loaded instantly — all charts, PRs, and metrics populate.
  3. When you're ready to start fresh, go to Profile → Settings → Reset all data.

⚙️ App preferences

  1. Sounds — toggle all in-app sounds (rest timer bell, PR chime) on or off under Profile → Settings → App → Sounds.
  2. Keep screen on — prevents your screen from sleeping while a workout is active. Toggle under Profile → Settings → App → Keep screen on during workout.
  3. Day starts at — controls when the streak counter rolls over to the next day. Default is 4 AM. Tap the tile to change it.

🐛 Reporting a bug

Go to Profile → Settings → Support → Report a bug. This opens a pre-addressed email in your default mail app. Describe what happened and tap Send — your report goes straight to the developer.
